# SferaTC Telegram Mini App - Новая архитектура

## Обзор

Это переписанная версия Telegram Mini App с улучшенной архитектурой, лучшей поддержкой и более простым кодом, при этом сохранившая все элементы дизайна оригинального приложения.

## Основные улучшения

1. **Правильное центрирование меню** - устранена проблема с неправильным позиционированием
2. **Модульная архитектура** - каждый компонент в отдельном модуле
3. **Улучшенная производительность** - оптимизированные стили и скрипты
4. **Лучшая расширяемость** - легко добавлять новые функции и модули
5. **Современные практики** - ES6+ модули, CSS Grid/Flexbox, событийная архитектура

## Особенности реализации

### 1. Центрирование

Используется единый подход к центрированию через CSS Grid:

```css
.app {
  display: grid;
 place-items: center;
  min-height: 100vh;
  min-height: 100dvh;
}
```

### 2. Модульная архитектура

- `app.js` - главный модуль, инициализирует все остальные
- `state.js` - управление состоянием приложения
- `eventSystem.js` - коммуникация между модулями
- `telegram.js` - интеграция с Telegram WebApp API
- `menu.js` - управление главным меню
- `sections.js` - управление контентными секциями

### 3. Сохранение дизайна

Все визуальные элементы оригинального приложения сохранены:
- Цвета и градиенты
- Шрифты Orbitron и Share Tech Mono
- Эффекты неоновой рамки
- Анимации и эффекты hover

## Запуск

Для запуска приложения в Telegram, разместите файлы в веб-доступное место и укажите URL в настройках Telegram Bot.

Для локального тестирования откройте `index.html` в браузере.

## Расширение функциональности

Для добавления новых секций:
1. Добавьте новый элемент в HTML
2. Обновите конфигурацию в `config.js`
3. При необходимости расширьте модули `menu.js` и `sections.js`

## Технологии

- Чистый HTML5/CSS3/JavaScript (ES6+)
- CSS Grid и Flexbox
- CSS Custom Properties
- ES6 модули
- Telegram WebApp API
