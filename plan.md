# План замены изображений на видео в боте SferaTC

## Текущая ситуация
В боте SferaTC используются изображения в следующих местах:
1. Приветствие - WELCOME_IMAGE_URL
2. Обучение - TRAINING_IMAGE_URL
3. Психолог - PSYCHOLOGIST_IMAGE_URL
4. Поддержка - SUPPORT_IMAGE_URL
5. Инструменты - TOOLS_IMAGE_URL
6. ChatGPT - CHATGPT_IMAGE_URL (переменная определена, но не используется в коде)

## Цель
Заменить все изображения на короткие видео без звука, которые автоматически проигрываются и зацикливаются.

## Требования к видео
- Формат: MP4
- Без звука (no audio track)
- Маленький размер (до 20MB для ботов)
- Короткая продолжительность (несколько секунд с зацикливанием)
- Автоматическое воспроизведение и зацикливание в Telegram

## Пошаговый план

### Этап 1: Подготовка
1.1. Создать вспомогательную функцию для отправки видео вместо изображений
1.2. Определить, какую из переменных CHATGPT_IMAGE_URL использовать (она определена в .env, но не используется в коде)

### Этап 2: Реализация для одного элемента
2.1. Добавить поддержку изображения в раздел ChatGPT, используя CHATGPT_IMAGE_URL
2.2. Создать функцию-заменитель для отправки видео вместо изображений
2.3. Заменить изображение в разделе ChatGPT на видео как тестовый пример
2.4. Протестировать изменения

### Этап 3: Массовая замена
3.1. После подтверждения работоспособности, заменить все остальные изображения на видео
3.2. Обновить все обработчики: приветствие, обучение, психолог, поддержка, инструменты

## Технические детали

### Вспомогательная функция
Создать функцию, которая будет:
- Проверять, доступен ли URL видео
- Отправлять видео вместо изображения с теми же параметрами (caption, reply_markup)
- Иметь fallback к текстовому сообщению при проблемах

### Методы Telegram API
- Вместо `reply_photo` использовать `reply_video`
- Вместо `send_photo` использовать `send_video`
- Для InputMedia использовать `InputMediaVideo` вместо `InputMediaPhoto`

### Параметры для видео
- `supports_streaming=True` - для лучшей совместимости
- `duration`, `width`, `height` - опционально для лучшей отрисовки
